# Summary

This directory contains summary documents and overviews.

## Purpose

Store documentation for:

- Project summaries
- Feature overviews
- Sprint retrospectives
- Release notes
- Changelog summaries

## Naming Convention

Use descriptive, kebab-case filenames:
- `feature-summary.md`
- `release-vX.X.X-summary.md`
- `sprint-XX-retrospective.md`

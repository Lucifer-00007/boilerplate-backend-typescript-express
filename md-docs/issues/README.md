# Issues

This directory contains documentation related to bugs, issues, and their resolutions.

## Purpose

Store documentation for:

- Bug reports and investigations
- Issue tracking notes
- Debugging logs
- Problem analysis and solutions

## Naming Convention

Use descriptive, kebab-case filenames with optional date prefix:
- `YYYY-MM-DD-issue-description.md`
- `bug-component-name.md`
- `issue-feature-name.md`

# Others

This directory contains miscellaneous documentation that doesn't fit into other categories.

## Purpose

Store documentation for:

- Meeting notes
- Brainstorming sessions
- Temporary documentation
- Experimental features
- Miscellaneous notes

## Naming Convention

Use descriptive, kebab-case filenames:
- `topic-name.md`
- `notes-subject.md`
- `misc-description.md`
